# Examen-Final-IA

## Como ejecutar el código para hacer pruebas con imágenes  
1. El modelo ya está entrenado por lo que este paso se puede saltar. En el directorio principal donde están los archivos con el 
código, ejecutar el entrenamiento con python 01_Entrenamiento_CNN.py en la terminal y esperar a que acabe.
2. El paso anterior generará el archivo "ocr_final_iker.h5".
3. Comprobamos que tenemos las imágenes en formato .png en nuestro directorio principal.
4. Ejecutamos el reconocimiento con py 02_Software_OCR_Principal.py en la terminal y nos pedirá el nombre de la foto, escribes
"nombre_foto.png" y se abrirá una ventana con el reconocimiento de los caracteres y se mostrará el resultado por terminal.
5. Cerrar la ventana de python y continuar escribiendo el nombre de todas las fotos hasta terminar.
